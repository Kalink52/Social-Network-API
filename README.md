## Installation
1.  Install Node.js 
2.  Install mongodb 
3.  Clone https://github.com/Kalink52/Social-Network-API
4.  cd into the cloned directory
5.  run "npm run seed"
6.  run the program with "node server.js"
## Usage
Api will be avaible at localhost:3001
